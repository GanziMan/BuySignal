# ⚙️ How to Start

## 0️⃣ Prerequisites

- [Node.js 18.20.4](https://nodejs.org/en/download/package-manager/)
- [npm 10.7.0](https://www.npmjs.com/package/npm/v/10.7.0)

## 1️⃣ Installation

### Front-End

```bash
$ cd client
$ npm install
```

## 2️⃣ Running the app

````
### Front-End
```bash
# development
$ npm run dev

# production mode
$ npm start
````
